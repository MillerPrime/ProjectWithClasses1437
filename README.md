# Project with a class - COSC 1437

## What it does
This program allows the user to enter quiz scores and calculates:
- The number of scores entered
- The highest score
- The lowest score
- The average score  

It validates input so only scores between 0 and 100 are accepted.  
The user can stop entering scores by typing `-1`.

## How to run it
1. Open the project in your Java IDE.
2. Compile the `MillerQuizScore.java` file.
3. Run the program.
4. Enter quiz scores when prompted.  
   - Type `-1` to stop entering scores.
5. The program will display the results in the console.
